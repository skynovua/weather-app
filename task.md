## **Тестове завдання: React Front-End Developer** 

**Завдання:** Створити React-застосунок, який показує актуальну погоду для обраного міста. 

Користувач повинен мати змогу: 

- ввести назву міста 

- отримати поточну погоду через API 

- переглянути основні погодні показники 

- додати місто в список обраних 

- перемикатися між збереженими містами 

- бачити стани loading, error, empty 

## **Основний функціонал:** 

## _1. Пошук міста:_ 

Реалізувати поле вводу для пошуку міста. 

Після введення назви міста застосунок має: 

- зробити запит до погодного API 

- відобразити результат 

- показати помилку, якщо місто не знайдено або запит не вдався 

## _2. Блок поточної погоди:_ 

Для обраного міста відобразити: 

- назву міста 

- температуру 

- погодний стан 

- іконку погоди 

- вологість 

- швидкість вітру 

- відчутну температуру 

## _3. Обрані міста:_ 

Користувач може: 

- додати місто в обрані 

- видалити місто з обраних 

- швидко відкривати погоду для міста зі списк 

Зберігання обраних міст: 

- у localStorage 

## _4. Додатковий блок_ 

Додати один із варіантів: 

- прогноз на кілька годин, або 

- прогноз на 3–5 днів 

## **Робота з API:** 

Можна використати одне з API: 

- OpenWeather API 

- WeatherAPI 

- Visual Crossing 

- будь-яке інше публічне погодне API
