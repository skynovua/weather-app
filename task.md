## **Тестове завдання: React Front-End Developer**

**Завдання:** Створити React-застосунок, який показує актуальну погоду для обраного міста.

Користувач повинен мати змогу:

- [x] ввести назву міста

- [x] отримати поточну погоду через API

- [x] переглянути основні погодні показники

- [x] додати місто в список обраних

- [x] перемикатися між збереженими містами

- [x] бачити стани loading, error, empty

## **Основний функціонал:**

## _1. Пошук міста:_

Реалізувати поле вводу для пошуку міста.

Після введення назви міста застосунок має:

- [x] зробити запит до погодного API

- [x] відобразити результат

- [x] показати помилку, якщо місто не знайдено або запит не вдався

## _2. Блок поточної погоди:_

Для обраного міста відобразити:

- [x] назву міста

- [x] температуру

- [x] погодний стан

- [x] іконку погоди

- [x] вологість

- [x] швидкість вітру

- [x] відчутну температуру

## _3. Обрані міста:_

Користувач може:

- [x] додати місто в обрані

- [x] видалити місто з обраних

- [x] швидко відкривати погоду для міста зі списк

Зберігання обраних міст:

- [x] у localStorage

## _4. Додатковий блок_

Додати один із варіантів:

- [ ] прогноз на кілька годин, або

- [x] прогноз на 3–5 днів
